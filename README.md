# DoShare - 大学生数据集导航平台

## 项目简介
    本项目旨在构建一个服务于本科生的数据集导航与学习共享平台。当前，本科生在课程学习与项目实践中普遍面临数据集来源分散、结构复杂、缺乏使用指导等困难。本平台针对这些问题，系统整合多领域高质量数据集，并提供清晰的分类导航与检索功能。平台为每个数据集配备字段说明、应用场景介绍及基础分析案例，帮助学生理解数据，并且建立数据集质量评分系统，根据建立的知识图谱为用户推送优质资源。同时，平台集成智能问答模块，可响应学生的自然语言提问，提供分析思路与代码示例。此外，用户社区支持经验分享与互助答疑。该平台致力于降低数据使用门槛，提升本科生的数据获取与应用能力，为学术训练与创新实践提供支持。
